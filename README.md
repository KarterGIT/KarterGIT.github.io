# KarterGIT.github.io

Personal portfolio site for **Karter Riggs** — Artificial Intelligence student & builder.

🌿 Live site: https://kartergit.github.io

## Built with

- Hand-written HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- Natural sage-green theme with light/dark mode
- Fully responsive, accessible, and fast

## Local preview

Just open `index.html` in a browser, or serve the folder:

```bash
# Python
python -m http.server 8000
# then visit http://localhost:8000
```

## Structure

```
index.html      # all page content
styles.css      # theme + layout
script.js       # theme toggle, menu, scroll reveals
*.pdf           # résumé + certificates (linked from the page)
```
